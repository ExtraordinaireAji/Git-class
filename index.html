introduction to Git

Git is an open source version control system that is used to track changes made to our code

advantages of using Git

  • speed and performance
  • distributed version control
  • collaboration
  • conflict resolution
  • history and traceability
  • branching and merging

basic git command

  1. git init: this is used to initialize a git repository
  2. git status: this is used to check the status of our file
  3. git add . :this is used to track the changes made to our file
  4. Git commit -m "the actual commit message": this is used to save a snapshot of our file using a descriptive caption
  5. Git remote add origin: this is used to connect a local repository to a remote repository on github
  6. Git push origin main: this is used to push code from local to remote repository
